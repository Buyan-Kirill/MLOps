# MLOps
Проект, выполненный в рамках задания по курсу MLOps

В рамках проекта предполагается создать рекомендательную систему, которая по введённому пользователем названию произведения и автору должна предложить другие произведения, которые могут быть ему интересны.

В качестве датасета используется Goodreads Book Datasets With User Rating 2M (https://www.kaggle.com/datasets/bahramjannesarr/goodreads-book-datasets-10m/data). В нём есть таблицы с различными книгами и их характеристиками, а также таблицы с отзывами пользователей о различных книгах.

Для решения задачи информация о произведение будет переводиться в информативное веторное пространства и далее рекомендация пользователю будет производиться на основание близости к заданному пользователем книги, а также общем рейтинге книги (то есть расстояние с дополнительным шкалирующим параметром - рейтингом).

Для оценки качества модели предполагается использовать оценки пользователей. В силу наличия только отзывов пользователей о книгах, для каждого пользователя будет смотреться, насколько положительно оценённые произведения хорошо класетризованы и отделены от отрицательно оценённых в плане введённого выше расстояния.
